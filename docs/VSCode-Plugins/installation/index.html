安装

前置要求

  • VS Code 版本：1.88.0 及以上。
  • 支持平台：Windows / macOS / Linux（x86_64、aarch64、riscv64 桌面环境）。
  • 网络：可访问github或镜像站点以同步 Ruyi 软件包。
  • RuyiSDK 本体：未安装时插件会引导一键安装；也可手动完成安装与配置：参见 安装。

获取插件

  • VS Code 扩展市场搜索 “RuyiSDK” 直接安装。
  • 离线方式：从 https://github.com/ruyisdk/ruyisdk-vscode-extension/releases 下载最新 .vsix，在 VS Code 中选择 “从 VSIX 安装”。

首次使用

  1. 点击侧边栏 Ruyi 图标进入插件主页。
  2. 如未安装 ruyi，按提示一键安装或在命令面板执行 Ruyi: Detect Ruyi Installation。
  3. 在 Packages 视图安装所需工具链（如 gnu-upstream）。
  4. 在 Virtual Environments 视图点击 + 选择 Profile 和工具链创建虚拟环境，并点击名称激活。

设置项

  • ruyi.checkForUpdates：运行检测命令时自动检查 Ruyi 更新（默认 true）。
  • ruyi.ruyiPath：自定义 Ruyi 可执行路径（留空自动检测）。
  • ruyi.telemetry：是否启用遥测数据收集（未设则首次使用时询问）。

更多概念与 CLI 行为请参考 Ruyi 包管理器。
