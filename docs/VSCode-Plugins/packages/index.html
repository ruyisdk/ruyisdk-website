管理 Ruyi 软件包

插件的 Packages 视图提供可视化的包管理：

  • 分类浏览：包按类别分组显示；展开包名查看全部版本。
  • 搜索与刷新：工具栏按钮支持名称、类别筛选并刷新列表。
  • 安装/卸载：与 CLI 行为一致，操作后状态实时更新；已安装版本显示绿色勾选。

说明：插件的包来源、命名、版本规范等与 CLI 完全一致，详细功能说明请参见 Ruyi 包管理器。

常见操作

  • 安装工具链：在列表选中如 gnu-upstream 的目标版本，点击安装。
  • 卸载工具链：在已安装版本上点击卸载。
  • 刷新列表：点击工具栏刷新或命令面板 Ruyi: Refresh Package List。

若网络受限，请先在终端对 Ruyi 配置镜像源，再回到插件执行安装与更新。CLI 配置方法参考 安装。
