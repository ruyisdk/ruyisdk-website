RuyiSDK VSCode 插件

官方 VS Code 扩展，把 Ruyi 包管理器的能力以可视化方式嵌入 IDE，用于安装/更新工具链、创建虚拟环境、查看新闻、提取源码包等。

📄️ 功能概览

- 软件包管理：在 VS Code 侧边栏浏览/搜索/安装/卸载 Ruyi 软件包。

📄️ 虚拟环境（Virtual Environments）

📄️ 源码包提取（Extract RuyiSDK Package）

在 VS Code 资源管理器中右键目标文件夹，选择 Extract RuyiSDK Package，即可直接下载并解包官方示例或源码包。
