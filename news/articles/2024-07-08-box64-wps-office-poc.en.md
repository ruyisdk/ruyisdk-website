# Early Access: Using Box64 to Run WPS Office on RISC-V Systems

Thanks to the recent progress in the Box64 RISC-V porting effort, it is now possible to run common x86 binaries such as WPS Office on RISC-V Linux desktop distributions.
RuyiSDK is working on integrating this workflow; by following the steps in the article, you can get an early taste of office software on RISC-V.
