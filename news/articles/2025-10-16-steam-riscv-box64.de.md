# Gute Nachrichten: Steam kann jetzt stabil auf RISC-V laufen! Box64 liefert erneut ab

![Steam RISC-V Box64](/img/newsshowcase/2025-10-16-steam-riscv-box64.png)

Box64 Version v0.3.8 wurde veröffentlicht. Dieses Update bringt eine große Anzahl an Kompatibilitätskorrekturen für das RISC-V-Backend,
 wodurch Linux Steam (endlich) stabil ausgeführt werden kann!
