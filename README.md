# RuyiSDK Website ![example workflow](https://github.com/ruyisdk/ruyisdk-website/actions/workflows/deploy.yml/badge.svg) [![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](CONTRIBUTING.md)

Official website of RuyiSDK Project.

This repo contains the website configuration and documentation powering the [RuyiSDK website](https://ruyisdk.org/).

## ‚úàÔ∏è Getting started

### Prerequisites

1.  Git
2.  Node
3.  pnpm
4.  A fork of the repo _(for any contributions)_.

### Installation

1.  `cd ruyisdk-website` to go into the project root.
2.  Run `pnpm ci` to install the website's workspace dependencies.
3.  Run `pnpm run build` to build the website.

### Running locally

1.  `pnpm start` to start the development server _(powered by [Docusaurus](https://v2.docusaurus.io))_.
2.  Or `pnpm run serve` to start the server after `pnpm run build`
3.  Open http://localhost:3000/ site in your favorite browser.
