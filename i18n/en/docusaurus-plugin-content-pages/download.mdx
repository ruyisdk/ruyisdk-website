# You can get pre-built binaries from [GitHub releases](https://github.com/ruyisdk/ruyi/releases) or [RuyiSDK mirror](https://mirror.iscas.ac.cn/ruyisdk/ruyi/testing/)

## Download

```bash
$ wget https://mirror.iscas.ac.cn/ruyisdk/ruyi/testing/ruyi.amd64.20231211  #Download ruyi from the software source
$ chmod +x ./ruyi.amd64.20231211  #Add execution permissions to ruyi
$ sudo cp ruyi.amd64.20231211 /usr/local/bin/ruyi #Rename the ruyi.amd64.20231211 to ruyi and put it in PATH
$ ruyi version #Success！
```
