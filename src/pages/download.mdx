## 下载

```bash
$ wget https://mirror.iscas.ac.cn/ruyisdk/ruyi/releases/0.15.0/ruyi.amd64  #从软件源下载ruyi本体
$ chmod +x ./ruyi.amd64  #给ruyi添加执行权限
$ sudo cp ruyi.amd64 /usr/local/bin/ruyi #将ruyi本体改名为ruyi，并放入PATH
$ ruyi version #成功！
```
