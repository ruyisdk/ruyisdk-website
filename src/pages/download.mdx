<h2 align="center">关于 RuyiSDK 下载</h2>

RuyiSDK 的核心是一套完整的 RISC-V 开发工具链，包含编译器、调试器、模拟器等关键组件。我们推荐通过 **包管理器** 来获取和管理这些工具——它是 RuyiSDK 的基础工具，可一键安装不同版本的工具链、模拟器等工具。此外，如果你习惯使用集成开发环境，我们还提供 **VSCode 插件** 与 **Eclipse 插件**，在包管理器能力之上提供图形化操作界面，让工具链调用与环境管理更加直观，同时还提供了更多提升开发易用性的功能。

请按照以下方式开始下载使用 RuyiSDK：

### 第一步：确认操作系统

RuyiSDK包管理器当前支持 Linux 平台（x86_64 / arm64 / riscv64）。Windows 或 macOS 用户请先准备 Linux 环境（如虚拟机、WSL 或容器）。

> ⚠️ **请注意**：VSCode/Eclipse 及插件虽可运行于 Linux、Windows、macOS，但插件所调用的 ruyi 包管理器及其分发的工具链均需在 Linux 下运行，故推荐使用 Linux 系统。具体支持的 Linux 发行版类型与版本详见《<a href="https://ruyisdk.org/docs/Other/platform-support">RuyiSDK 的平台支持情况</a >》
