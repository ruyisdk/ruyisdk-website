# RuyiSDK at the 2026 ISCAS “Software Defines the Future” Public Science Day

On May 16, the Institute of Software, Chinese Academy of Sciences will host its 2026 “Software Defines the Future” Public Science Day at the Zhongguancun Software Park campus in Beijing. As an important contributor to the RISC-V software ecosystem, RuyiSDK will present its complete toolchain, operating system support, and developer resources, and will exchange ideas with developers, open source enthusiasts, university faculty, and students.

## Event Overview

This year’s Public Science Day is themed “Software Defines the Future” and includes visits to the Software Technology Museum exhibition hall, interactive science projects, and a popular science book exhibition for the *Turing Universe* series. The interactive project “Small Architecture, Great Possibilities: RISC-V Compiler Toolchains, Operating Systems, and Software Ecosystem” is planned and led by the RuyiSDK team at ISCAS. It is designed to show the public the full-stack ecosystem being built around RISC-V, from compilers and toolchains to operating systems and application software.

## RuyiSDK Product Overview

RuyiSDK is a full-stack software development kit for the RISC-V architecture. It aims to lower the barrier to RISC-V development and help the RISC-V software ecosystem thrive. The project includes core components such as the ruyi package manager, the RuyiSDK integrated development environment, and development plugins, covering the complete development path from low-level compilers to upper-level applications. Whether you are a RISC-V developer, researcher, or learner interested in open chip technology, the following materials can help you learn more about RuyiSDK.

RuyiSDK welcomes every developer to join us in building a new RISC-V software ecosystem.
